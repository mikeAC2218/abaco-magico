¿Cómo funciona?

Sumas y restas: Usa el ábaco para resolver problemas matemáticos. Selecciona la operación (suma o resta), ingresa los números y calcula el resultado.

Comprobar resultado: Después de calcular, puedes usar el ábaco para representar tu respuesta y comprobar si es correcta.

Mostrar en ábaco: El botón "Mostrar en ábaco" te ayuda a visualizar el resultado correcto en el ábaco para que puedas comparar con tu respuesta.

Sistema de acarreo: Cuando activas la primera bolita de una magnitud superior (ej: decenas), todas las bolitas de las magnitudes inferiores (unidades) se regresan al lado izquierdo automáticamente.

Cada fila representa un valor diferente: unidades, decenas, centenas, etc. El número total se calcula sumando los valores de todas las filas.
